Convert .gpx to Google Maps

Upload a .gpx file and convert it to directions in Google Maps.

FAQ

  • What is this? A tool to very roughly convert .gpx files into Google Maps Directions. I wanted to be able to take routes from Strava and navigate through them using Google Maps instead of Strava's own route navigation and couldn't find one which did just that.
  • Why is Google Maps not following the exact path? The entire path from the .gpx file is reduced down to a few (hopefully) key waypoints to make the navigation 1) bearable and 2) not overwhelming for the navigation client (Google Maps doesn't allow manually using more than 10 waypoints as of 2022-07-25). The tool can be configured to use at most 3-30 with no guarantee all will be used. Play around with the Max precision to see if you can find something that works for you! I've found 25 works pretty well on a 50km bike ride.
  • Why does the image look the same when I change vehicle type? The image only shows the waypoints Google Maps will navigate to, it doesn't actually show the path that will be taken.
  • Do you keep my files when I upload them? No, I do not. In fact, you can see for yourself on my Github. The uploaded files are only read from during the upload, they're never stored and I see no reason to do so on my end.
  • How do I export the .gpx file from Stava? They have an article on that explaining how to export them on their support page.
